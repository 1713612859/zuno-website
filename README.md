# ZUNO Corporate Website

A complete React, TypeScript, Vite, and Tailwind CSS website for ZUNO, an integrated business technology provider focused on hardware distribution, software solutions, and payment solutions.

## Pages

- Home
- About
- Products & Services
- Solutions
- Partners
- Support
- Contact

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- lucide-react icons

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

On Windows PowerShell, if `npm` is blocked by script execution policy, use:

```bash
npm.cmd install
npm.cmd run dev
```

The project includes a local `.npmrc` using the Taobao npm mirror:

```bash
registry=https://registry.npmmirror.com
```

## Build

```bash
npm run build
```

Windows PowerShell alternative:

```bash
npm.cmd run build
```

## Notes

- The contact form is a static UI and does not submit to a backend.
- Contact information uses placeholders:
  - Email: contact@zuno.example
  - Phone: +63 XXX XXX XXXX
  - Location: Philippines
- No customer numbers, awards, partner logos, certifications, or brand names are fabricated.
